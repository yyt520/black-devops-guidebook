# 远程 git remote

> Manage set of tracked repositories
>
> 管理一组跟踪的存储库。

```bash
# 查看远程分支
git remote

# 列出远程分支的详细信息
git remote -v

# 查看指定远程仓库信息
git remote show <remote>

# 添加新的远程仓库
git remote add orgin <shortname> <url>

# 获取远程仓库变化
git pull <remote> <branch>
```
