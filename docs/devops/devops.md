DevOps 从需求到设计到开发、到测试到运维，甚至是线上的运营反馈整个全生命周期的，所以它应该是一个打通多个部门协调，协作的这样一个平台。至于工具和自动化实际上只是 DevOps 实现的一种手段。或者说 DevOPs 是通过工具，自动化，来达到这种通过工具链与持续集成、交付、反馈、优化进行端到整合，完整无缝的跨团队、跨系统协作。
